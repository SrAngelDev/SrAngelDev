## 🌟 **Proyectos Relevantes**

### 1. **Mi Trabajo Final de Grado - NexoPostal**

**Plataforma integral de logística y paquetería nacional (2025)**
▶️ Arquitectura de microservicios con 3 SPAs Angular + 5 servicios .NET, bases de datos independientes y pipeline CI/CD completa.
✅ **Logros clave:**

* Diseño y desarrollo **full stack** de plataforma end-to-end: desde cotización → pago → admisión → operativa → reparto → tracking
* **Frontend:** 3 aplicaciones Angular especializadas (clientes, intranet, drivers) con Tailwind CSS, SignalR, mapas Leaflet
* **Backend:** ASP.NET Core 10 con arquitectura de microservicios, 7 roles con permisos granulares, patrón ROP (Result<T>)
* **Base de Datos:** PostgreSQL 16 con instancias independientes por servicio (Database per Service)
* **Integraciones:** Stripe Checkout (pagos), SMTP, notificaciones en tiempo real con SignalR, GPS tracking
* **DevOps:** Docker, Docker Compose, Nginx, GitHub Actions, GHCR — pipeline CI/CD automatizada con tests (xUnit, Testcontainers, Playwright)
* **Estado:** ✅ En producción en [nexopostal.es](https://nexopostal.es) con 100% de cumplimiento en plazos
* 📦 **Monorepo estructurado** con 8 módulos, cobertura de tests unitarios, integración y E2E
* Licencia MIT — [Repositorio GitHub](https://github.com/SrAngelDev/NexoPostal)
